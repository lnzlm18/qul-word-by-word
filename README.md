# QUL Word-by-Word Reader

Interactive word-by-word Quran reader with transliteration and grammar colour-coding — built on **Tarteel's [Quranic Universal Library (QUL)](https://qul.tarteel.ai)**.

## QUL Resources Used
| Resource | QUL Category |
|---|---|
| Word-by-word Arabic | Quran Scripts |
| Word translations | Translations → Word by Word |
| Transliteration | Transliteration → Word by Word |
| Grammar / part of speech | Quranic Grammar and Morphology |

## Getting Started
```bash
npm install
npm run dev
```
